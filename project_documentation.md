# 🤖 Agents Unleashed — Official Application Documentation

**A fully autonomous multi-agent developer workspace powered by CrewAI-inspired architecture, Vite, NextJS, Python, and Supabase.**

---

## 🏗️ System Architecture Overview

_Agents Unleashed_ is a production-grade development assistant. Unlike standard conversational bots, it utilizes a structured pipeline of 8 specialized LangGraph agents. The user simply inputs raw "Requirements," and the agents autonomously debate, plan, code, test, and document the entire requested project from bare metal up.

### Technology Stack

1. **Frontend**: React 18, Vite, TypeScript, Zustand, TailwindCSS, Framer Motion
2. **Orchestrator Backend**: Python 3.10+, FastAPI, LangChain, LangGraph, Uvicorn
3. **Database Layer**: MongoDB (Project State), Supabase PostgreSQL w/ PGVector (Agent Memory Retrieval)
4. **LLM Inference Routing**: Google Gemini (Primary), OpenAI GPT-4o-mini (Fallback 1), Anthropic Claude 3.5 Sonnet (Fallback 2)

---

## ⚙️ AI Agent Pipeline

The core capability of this project is the **Python Orchestrator** which connects specialized LLM personas using LangGraph execution loops.

| Agent Name               | Functionality                                                                                                                                |
| :----------------------- | :------------------------------------------------------------------------------------------------------------------------------------------- |
| **`RequirementAgent`**   | Parses natural language into structural MVP features, goals, constraints, and JSON specifications.                                           |
| **`ArchitectureAgent`**  | Receives the structured requirements and designs file trees, technology stacks, and database layouts.                                        |
| **`CodeGenAgent`**       | Synthesizes massive, highly detailed blocks of code targeting maximum production-ready coverage per file.                                    |
| **`DebugAgent`**         | Analyzes the generated code for security vulnerabilities, syntax bugs, and performance impacts.                                              |
| **`CritiqueAgent`**      | Evaluates the entire project holistically and assigns a Quality Score (0-10). Can trigger full re-writes if the codebase fails CI standards. |
| **`TestingAgent`**       | Auto-generates unit tests explicitly targeting the behavior generated by the `CodeGenAgent`.                                                 |
| **`DocumentationAgent`** | Emits structural READMEs, deployment instructions, and API references.                                                                       |
| **`MemoryAgent`**        | Extracts global insights into a Supabase PGVector database to remember coding habits long-term.                                              |

## 🛠️ API & Fallback Systems

To guarantee uptime, the LangGraph backend operates using a custom `SyncRequestsGeminiModel`.
Due to rate-limit toxicity affecting the synchronous `httpx` threading loop internally on Windows, _Agents Unleashed_ utilizes pure HTTP `requests` bypassing deadlocks.
If Google Gemini throws a `429 Quota Exceeded` exception, the framework immediately steps to the next API key down the `.env` chain. If all Google Keys fail, it attempts an OpenAI endpoint, then finally an Anthropic endpoint.

## 🎨 UI/UX Specifications

The user interface features a dual-pane workspace:

- **Left Panel (Agent Feed)**: A live EventSource websocket streams the internal thoughts and execution boundaries of all 8 agents as they process the workflow.
- **Right Panel (Code Preview)**: An interactive viewer built with `react-syntax-highlighter` mapping Zustand's state. When a user clicks any module emitted by an agent on the left, it instantly renders on the right (JSON architecture models, Markdown documents, Code blocks, and Debug traces).
